# registrProg
Система учёта жильцов и квартир
Консольное приложение на языке C++, предназначенное для управления базой данных жильцов и квартир в жилом комплексе.

Возможности программы
Жильцы: Добавление и удаление жильцов, просмотр общего списка.

Квартиры: Добавление и удаление квартир, привязка к этажу и количеству комнат.

Взаимосвязи: Заселение жильцов в конкретные квартиры и их выселение.

Поиск: Вывод списков квартир по этажам или количеству комнат.

Файлы: Автоматическое сохранение данных в файлы и загрузка при запуске.

Структура проекта
source.cpp — основной файл с меню и логикой программы.

fileManager.cpp / .h — работа с файловой системой (загрузка и сохранение).

functions.cpp — основные функции обработки данных.

struct.h — описания структур (Resident, Apartment).

prototypes.h — заголовки всех функций.

Запуск
Убедитесь, что файлы apartments.txt и residents.txt находятся в той же папке, что и исполняемый файл .exe.

Соберите проект через Visual Studio (Ctrl + Shift + B).

Запустите файл (Ctrl + F5).

Формат данных
Программа работает с двумя текстовыми файлами:

residents.txt — содержит список жильцов.

apartments.txt — содержит параметры квартир и ID проживающих в них людей.

Примечание: Если файлы отсутствуют, программа создаст пустую базу при первом сохранении.
